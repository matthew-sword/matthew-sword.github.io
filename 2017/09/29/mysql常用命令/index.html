​ 登录退出mysql

  • mysql -v 版本

  • mysql -u root 用户

  • mysql -p 密码

  • mysql -P 端口

  • mysql -h 服务器地址

  • exit quit \q 退出服务器

    ​

    修改提示符

  • mysql -uroot -ppwd – prompt 提示符 登陆前，修改mysql提示符

  • prompt 提示符 登陆后，修改提示符

  • 提示符常用参数 \D –完整日期 \d –当前数据库 \h –服务器名称 \u –当前用户

    ​

    查看常用信息

    • SELECT VERSION() 显示mysql版本

    • SELECT NOW() 显示当前时间

    • SELECT USER() 显示当前用户

    • 书写规范 关键字和函数大写 有关数据的小写

    ​

    数据库操作

    • CREATE {DATABASE | SCHEMA} [IF NOT EXISTS] db_name [DEFAULT] CHARACTER SET [=] charset_name

      ​ ——创建数据库{}必选项 []可选项 | 从其中任选其一

    • SHOW {DATABASES | SCHEMAS} [LIKE ‘pattern’ | WHERE expr] 查看数据库

    • SHOW WARNINGS 查看警告

    • SHOW CREATE DATABASE db_name 显示创建数据库时的信息

      ​

      修改数据库

    • ALTER {DATABASE | SCHEMA} [db_name][ DEFAULT] CHARACTER SET [=] charset 修改数据库编码方式

    • DROP {DATABASE | SCHEMA} [IF EXISTS] db_name 删除数据库

数据类型

  • 整数
  • 浮点型 FLOAT(8,2)—总共8位小数点后2位
  • 时间日期型
  • 字符型

创建数据表

行：记录
列：字段 column

  • USE db_name 打开数据库
  • SELECTE DATABASE() 显示当前打开的数据库
  • CREATE TABLE [IF NOT EXISTS] tb_name( col_name datatype, …) 创建数据表
  • SHOW TABLES [FROM db_name ][LIKE ‘pattern’ | WHERE expr] 查看数据表
  • SHOW TABLES FROM mysql 查看整个mysql中的数据表

操作数据表

  • SHOW COLUMNS FROM tb_name 查看数据表结构

  • INSERT [INTO] tb_name [col_name, …] VALUE (value, …) 添加数据，可以只添加一部分数据

  • SELECT expr,… FROM tb_name 查找数据

  • col关键字 AUTO_INCREMENT 自动编号，只能与主键组合使用，默认增量为1

  • PRIMARY KEY 主键，一张表必须有一个主键，不能为空

  • UNIQUE KEY 唯一约束 ，一张表可以有多个，可以为null

  • DEFAULT 默认约束，对记录的默认值

INSERT

  • INSERT [INTO] tb_name [col_name] {VALUES | VALUE} (,,,,),(,,,,),(,,,,)….插入多条记录

  • INSERT [INTO] tb_name SET col_name = {(expr) | DEFAULT} 使用子查询
