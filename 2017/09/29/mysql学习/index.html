mysql学习

2017-09-29 mysql

#基础命令

​ 命令以 “;”结束

  • 退出mysql控制台：exit/quit

  • 创建数据库：creat database db_name; 规则：以字母、数字、下划线、$组成，不能单独使用数字，最长64个字符

  • 选择数据库：use db_name;

  • 查看数据库：show databases;

  • 删除数据库：drop db_name;

#数据类型

数字类型

  • TINYINT 1字节

  • BIT 1字节

  • BOOL 1字节

  • SMALLINT 2字节

  • MEDIUMINT 3字节

  • INT 4字节

  • BIGINT 8字节

    ​

  • FLOAT 8/4字节

  • DOUBLE 8字节

  • DECIMAL 自定义长度

##字符串类型

普通文本

  • char(M) 最多255个字符
  • char 类似char(M)，最多255个字符
  • varchar(M) 长度可变

可变文本TEXT&BLOB

TEXT用于储存长文本，BLOB用于储存二进制数据，支持任何数据。如文本、声音、图像等。、

  • TINYBLOB 最大255字节

  • TINYTEXT 最大255字节

  • BLOB 最大65 535字节

  • TEXT 最大65 535字节

  • MEDIUMBLOB 最大16 777 215字节

  • MEDIUMTEXT 最大16 777 215字节

  • LONGBLOB 最大4 294 967 295字节

  • LONGTEXT 最大4 294 967 295字节

###特殊类型SET&ENUM

  • Enum(“value1”, “value2”, …)：最大字节数65 535，其值只能是所列值或者NULL
  • SET(“value1”, “value2”, …)：最大字节数64，其值只能是所列值或者NULL

##时间日期类型

  • DATE：格式 YYYY-MM-DD

  • TIME：格式 HH:MM:SS

  • DATETIME：格式 YYYY-MM-DD HH:MM:SS

  • TIMESTAMP：处理报告时所显示的格式

  • YEAR：格式 YYYY

#操作数据表

###创建数据表

create [TEMPORARY] table [IF NOT EXISTS] table_name [(definition,...)] [table_options] [select_statement];

  • TEMPORARY表示创建临时表
  • definition表示列表属性
  • table_options表的一些特性参数

definition说明：
每一列格式如下 col_name type[ not null | null][DEFAULT default_value] [PRIMARY KEY]

查看表结构

  • show columns查看列 ：show columns from db_name.table_name;

  • describe：describe table_name ; \ describe table_name columns; describe 可以简写为desc

###修改表结构

alter [IGNORE] table table_name 修改命令1,修改命令2,...; 其中IGNORE表示如果出现重复行，则只执行一行，其它重复行被删除

修改命令：

  • ADD [COLUMN] create_definition [FIRST/ AFTER column_name] 添加新列
  • ADD INDEX [index_name] (index_col_name,...) 添加索引
  • ADD PRIMARY KEY (index_col_name) 添加主键
  • ADD UNIQUE [index_name] (index_col_name) 添加唯一索引
  • ALTER [COLUMN] col_name 修改列名
  • CHANGE [COLUMN] old_name create_definition 修改字段类型
  • MODIFY [COLUMN] create_definition 修改子句定义字段
  • DROP [COLUMN] col_name 删除列
  • DROP PRIMARY KEY 删除主键
  • DROP [INDEX] index_name 删除索引
  • RENAME [AS] new_tb_name 更改表名

###重命名&删除表

  • rename table old_name to new_name; 重命名
  • drop table [if exists] tb_name; 删除

#操作数据表记录

编辑记录

  • 插入记录 insert into tb_name(col2,col2,col3...)values( value1,value2,...); 一条语句只能插入一行;
  • 更新记录 update tb_name set col1 = new_val1,col2 = new_val2..., where condition; 如果不指定condition则所有行记录都被修改。
  • 删除记录 delete from tb_name where condition; 如果不指定condition则所有行记录都被删除。
